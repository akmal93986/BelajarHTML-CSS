Belajar HTML Dasar

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Yuk kita pelajari tag-tag dasar dalam HTML!

1. Struktur Dasar HTML

Setiap dokumen HTML punya struktur dasar seperti ini:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    Konten di sini
  </body>
</html>

2. Tag Judul

Gunakan tag <h1> sampai <h6> untuk membuat judul.

<h1>Judul Besar</h1>
<h2>Judul Sedang</h2>

3. Paragraf

Untuk teks paragraf, gunakan tag <p>.

<p>Ini adalah paragraf.</p>

4. Gambar

Untuk menampilkan gambar, gunakan tag <img>:

<img src="gambar.jpg" alt="Deskripsi gambar">

5. Link

Untuk membuat link ke halaman lain:

<a href="https://google.com">Kunjungi Google</a>
