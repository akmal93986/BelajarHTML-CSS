Belajar JavaScript Dasar

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif. Misalnya: klik tombol, validasi form, animasi, dan lainnya.

1. Menambahkan JavaScript

JavaScript bisa ditulis di dalam tag <script>:

<script>
  alert("Halo dunia!");
</script>

2. Variabel

Digunakan untuk menyimpan data:

let nama = "Akmal";
let umur = 16;

3. Fungsi

Fungsi adalah blok kode yang bisa dipanggil berulang kali:

function sapa() {
  alert("Selamat datang di BelajarRPL!");
}
sapa();

4. Event Klik

Contoh interaksi dengan tombol:

<button onclick="alert('Kamu menekan tombol!')">Klik Aku</button>

5. Manipulasi DOM

DOM (Document Object Model) memungkinkan kita mengubah isi HTML secara langsung:

<p id="teks">Teks lama</p>
<script>
  document.getElementById("teks").innerText = "Teks baru dari JS!";
</script>
