Belajar PHP Dasar

PHP (Hypertext Preprocessor) adalah bahasa pemrograman sisi server yang digunakan untuk mengembangkan aplikasi web dinamis. Yuk kita mulai!

1. Struktur Dasar PHP

<?php
echo "Halo, dunia!";
?>

2. Variabel dan Tipe Data

<?php
$nama = "Akmal";
$umur = 17;
$tinggi = 1.7;
$aktif = true;
?>

3. Percabangan

<?php
if ($umur >= 17) {
  echo "Sudah cukup umur";
} else {
  echo "Masih di bawah umur";
}
?>

4. Perulangan

<?php
for ($i = 1; $i <= 5; $i++) {
  echo "Angka $i <br>";
}
?>

5. Form dan $_POST

<form method="post">
  Nama: <input type="text" name="nama">
  <input type="submit">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $nama = $_POST["nama"];
  echo "Halo, $nama!";
}
?>
