Belajar CSS Dasar

CSS (Cascading Style Sheets) adalah bahasa untuk mengatur tampilan elemen HTML. Dengan CSS, kita bisa mengubah warna, ukuran, posisi, dan banyak lagi.

1. Menyisipkan CSS

CSS bisa ditulis di tiga tempat:

  • Inline: langsung di tag HTML
  • Internal: di dalam tag <style> di head
  • Eksternal: file .css terpisah dan dihubungkan via <link>
<style>
  body {
    background-color: lightblue;
  }
</style>

2. Selektor

Digunakan untuk memilih elemen yang akan di-style:

p {
  color: red;
}

3. Properti Umum

Contoh properti CSS dasar:

h1 {
  font-size: 24px;
  color: blue;
  text-align: center;
}

4. Class dan ID

Kita bisa memberikan class atau id agar elemen lebih spesifik:

<p class="teks-merah">Ini teks merah</p>

.teks-merah {
  color: red;
}
